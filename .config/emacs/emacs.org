#+TITLE: CL  Emacs Configuration
#+PROPERTY: header-args:emacs-lisp :tangle yes :comments org

This is the literate source for =init.el=. Every =emacs-lisp= block
below with a live =:tangle= (the file-wide property line above turns
that on by default) gets written out to =emacs.el= and loaded by
=init.el= via ~org-babel-load-file~. Edit this file, not the
generated one.

** Tangle this file on save

Guarded so saving *this specific file* re-tangles =emacs.el=, but
saving any other Org file on the system doesn't. This keeps the
tangled file fresh for git diffs / anyone inspecting it directly,
even though =init.el= itself doesn't strictly need it thanks to
~org-babel-load-file~'s own mtime check at startup.

#+begin_src emacs-lisp
(defun cl/org-babel-tangle-config ()
  (when (string-equal (file-truename buffer-file-name)
                       (file-truename (expand-file-name "emacs.org" user-emacs-directory)))
    (let ((org-confirm-babel-evaluate nil))
      (org-babel-tangle))))

(add-hook 'org-mode-hook
          (lambda ()
            (add-hook 'after-save-hook #'cl/org-babel-tangle-config nil 'local)))
#+end_src
